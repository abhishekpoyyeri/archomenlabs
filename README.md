# ArchomenLabs & 0pinion

Welcome to the **ArchomenLabs** product studio repository. We are a product studio creating focused, thoughtful applications built for the people.

## 0pinion
**0pinion** is our first core product currently in development.

It is a **minimal, monochromatic debating platform** designed to foster real, thoughtful conversations. 

### Why 0pinion?
- **No Doomscrolling**: The platform is built around intentional engagement, not endless scrolling.
- **Zero Distractions**: No media, no images, no noise.
- **Pure Debate**: A platform dedicated strictly to sharing opinions and engaging in thoughtful debate.

## Tech Stack
This project is built using:
- **React**
- **Vite**

## Getting Started

To run the project locally:

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
